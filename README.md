# Task - 1 Project

This project is a web portfolio for Pushkar Mahadik, featuring:
- Responsive design with dark/light theme toggle
- Animated particle background
- Skills progress bars
- Projects showcase
- Contact form

## Features
- HTML, CSS, JavaScript frontend
- Particle animation effect
- Theme toggle (black/white with gradients)
- Responsive layout for all devices

## Folder Structure
- `Task -1/frontend/` — Main web app (index.html, style.css, script.js)
- `Task -1/assets/` — Images and icons

## How to Run
Open `Task -1/frontend/index.html` in your browser. No backend required.
